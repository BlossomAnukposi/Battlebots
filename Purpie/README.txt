This is a description of our BattleBots Purpie - number BB027.
Coded by Cristian Tifan & Jia Men Lam.

Purpie is responsible for the first part of the maze "Line Race".
It start moving with an object at the start once the flag is removed from it using Ultrasonic Distance sensor.
When Purpie start it should follow the black line and avoid other objects till it find the square to drop the object and stop,so the next BattleBots can start.
